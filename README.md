# Diablo Shadowdark — Homebrew RPG

**Versão:** 1.0 (Playtest)  
**Sistema:** Diablo Shadowdark  
**Autor:** Paulo Souza

## Sobre o Projeto

Este é um sistema de RPG de mesa inspirado na franquia **Diablo**, construído sobre as bases do Shadowdark. O objetivo é capturar a atmosfera sombria e visceral dos jogos digitais — mortais vulneráveis, poderes limitados e decisões rápidas — adaptada para a mesa de RPG.

## Princípios do Jogo

- **Velocidade:** Turnos rápidos, sem longas deliberações. Ideal para grupos de 5+ jogadores.
- **Desafio:** Emula a sensação de jogar Diablo no modo Hardcore.
- **Imersão:** Simula a franquia Diablo com ambientação sombria e narrativa visceral.

## Conteúdo

- `Diablo Homebrew RPG.docx` — Documento principal com regras, cenários e fichas.

## Como Usar

1. Baixe o arquivo `.docx`
2. Leia as regras na seção **Princípios do Jogo**
3. Use o **Template de Cenário** para criar suas próprias aventuras
4. O cenário de exemplo **"O Sarcófago Escarlate"** é um ótimo ponto de partida
