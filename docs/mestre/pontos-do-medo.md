# Pontos do Medo

O Mestre acumula **Pontos do Medo** durante a sessão e os gasta para aumentar a pressão sobre os heróis. Use dados ou cubos roxos como marcadores físicos na mesa — os jogadores veem o estoque crescer e sentem a tensão.

## Como Ganhar

| Gatilho | Pontos |
|---|---|
| Início de cada combate (renda base) | +1 |
| Crítico natural (d20 = 20) de qualquer jogador | +1 |
| Falha crítica (d20 = 1) de qualquer jogador | +1 |
| Jogador vacila ou enrola demais na decisão | +1 |
| Personagem cai inconsciente | +1 |
| Grupo limpa um encontro sem sofrer nenhum dano | +2 |
| Grupo ignora um gancho de missão óbvio | +1 |
| Grupo entra em um novo nível de dungeon | +1 |
| Grupo está separado *(+1 por turno enquanto durar)* | +1/turno |

## Como Gastar

### Reforços de Combate

| Custo | Reforço |
|---|---|
| 2 pts | Pack de 1d4 minions do mesmo tipo já presente na cena |
| 2 pts | Um inimigo derrotado levanta com metade do PV máximo |
| 3 pts | Emboscada — 1d3 inimigos surgem flanqueando o grupo |
| 3 pts | Revelação — um inimigo escondido na cena entra em combate |
| 3 pts | Um monstro fugido reaparece no pior momento |
| 4 pts | Mini-chefe de reforço entra em cena (use um monstro de ND+1) |

### Ações dos Monstros Fora de Turno

| Custo | Ação |
|---|---|
| 1 pt | Movimento extra — monstro se reposiciona sem gastar turno |
| 2 pts | Ataque extra — monstro faz um ataque imediato fora de turno |
| 2 pts | Habilidade especial — monstro usa uma habilidade do seu bloco de estatísticas |
| 2 pts | Recuo tático — monstro recua para posição vantajosa (cobertura, terreno elevado) |
| 3 pts | Sacrifício — um lacaio absorve dano destinado ao chefe (dano todo vai para o lacaio) |

### Armadilhas e Perigos Ambientais

| Custo | Perigo |
|---|---|
| 2 pts | Armadilha de dano — 1d6 Físico, Salvaguarda DES DC 13 para metade |
| 2 pts | Terreno traiçoeiro — área se torna difícil (custo dobrado de movimento) |
| 2 pts | Bloqueio — porta, portão ou escombros cortam o caminho de um personagem |
| 3 pts | Armadilha de controle — alvo fica preso (Salvaguarda FOR DC 13 para escapar no turno) |
| 3 pts | Perigo elemental — Fogo, Gelo ou Veneno, 1d8 dano, Salvaguarda CON DC 13 para metade |

### Pressão Direta sobre os Heróis

| Custo | Ação |
|---|---|
| 1 pt | Forçar um jogador a rerolar qualquer dado |
| 1 pt | Corromper uma cura (efeito reduzido à metade) |
| 3 pts | Negar o dano de um ataque dos jogadores |

### Salvaguarda Forçada

O Mestre gasta pontos para forçar uma Salvaguarda imediata em um personagem. O atributo e o efeito em caso de falha são escolhidos pelo Mestre conforme a situação.

| Custo | DC |
|---|---|
| 2 pts | DC 12 |
| 3 pts | DC 15 |
| 4 pts | DC 18 |

### Pressão Narrativa *(fora de combate)*

| Custo | Complicação |
|---|---|
| 1 pt | Mercador sem estoque de um item específico que o grupo precisa |
| 1 pt | Preço dobrado — o vendedor "reconhece o desespero" dos heróis |
| 2 pts | Tempo — algo que estava esperando chegou antes do previsto |
| 2 pts | Testemunha — alguém viu o que os heróis fizeram e pode ser um problema |
| 3 pts | Informação falsa — uma pista ou rumor que receberam estava errado |
| 3 pts | Traição — um aliado menor age contra os interesses do grupo no momento errado |

> **Dica — Informação Falsa:** Não revele que a informação era falsa — revele a verdade como uma descoberta nova. O informante pode ter acreditado no que disse (estava desatualizado ou foi enganado também). Use de preferência em informações de rota ou contexto, não no objetivo central da missão. Plante a dúvida antes de gastar o ponto: apresente a informação com linguagem incerta (*"pelo que ouvi dizer..."*) para criar espaço natural para a contradição depois.
