# Criaturas

Este apêndice contém os statblocks de todas as criaturas que aparecem nas tabelas de encontros do livro. As criaturas estão organizadas por bioma, espelhando as tabelas — quando você rolar um encontro, basta abrir a seção correspondente.

---

### Como ler um statblock compacto

> **PV** = Pontos de Vida | **CA** = Classe de Armadura | **Ataque** = bônus para acertar | **Vel.** = Velocidade de movimento | **PA** = Pontos de Ação por turno (mínimo 3 para todas as criaturas)

---

### Bônus e Bônus Fraco (para testes e salvaguardas)

Criaturas não têm atributos separados. Quando uma situação exigir um teste ou salvaguarda de uma criatura — ser derrubada, resistir a um feitiço, escapar de uma armadilha — use esta regra simples:

- **Bônus:** valor usado quando a criatura é boa naquilo. Geralmente igual ao bônus de ataque.
- **Bônus Fraco:** valor usado quando a criatura é ruim naquilo. Geralmente metade do Bônus, arredondado para baixo.

O Mestre decide qual se aplica baseado no que faz sentido para a criatura. Um Homem-Bode resiste bem a ser empurrado (usa o Bônus), mas foge mal de uma armadilha (usa o Bônus Fraco). Não há cálculo — só ficção.

Criaturas comuns têm apenas um valor de ataque, que serve como Bônus padrão. Chefes e criaturas únicas listam os dois valores explicitamente, junto com exemplos do que cada um cobre.

---

### Resistências (RD)

Criaturas com resistência usam o mesmo sistema da Matriz de Resistência dos jogadores — subtração direta do dano recebido. "RD 3 contra fogo" significa subtrair 3 do dano de fogo, não dividir pela metade.

---

### Loot

Todas as criaturas usam apenas **ouro (po)** como moeda. O loot listado é o que elas carregam; para tesouros mais elaborados, use as tabelas do Capítulo 6.
